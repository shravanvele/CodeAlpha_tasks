# Age Calculator

An easy-to-use **Age Calculator** web application that calculates a user’s exact age in **Years, Months, and Days** based on their Date of Birth.

This project is built using simple **HTML, CSS, and JavaScript** and is perfect for beginners to understand date handling and basic web development.

---

##  What This Project Does

In this project, we have created a small web application that:

- Takes the user’s **date of birth** as input  
- Validates the input (no empty value, no future date allowed)  
- Calculates the exact age by comparing:
  - Current date  
  - Birth date  
- Displays the result clearly in:
  - Years  
  - Months  
  - Days  
- Shows **error messages** if the input is invalid  
- Uses a clean and simple **UI design** for better user experience  

This project demonstrates:
- DOM manipulation using JavaScript  
- Date calculations in JavaScript  
- Basic form validation  
- Simple UI styling with CSS  

---

##  Features

- Instant age calculation  
- Accurate result in Years, Months & Days  
- Input validation for better reliability  
- Responsive and clean interface  
- Beginner-friendly project structure  

---

##  Technologies Used

- **HTML** – structure of the webpage
