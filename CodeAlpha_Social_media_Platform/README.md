## Project Overview  
This project is a lightweight social media web application. Users can sign up, view their profile, post updates, view posts by others, manage their profile information, and enjoy a responsive frontend experience.

## Features  
- User registration & login  
- Profile page with user information  
- Posting / updating content (text + optionally media)  
- Viewing posts from other users  
- Responsive UI (desktop + mobile)  
- Dark/light mode toggle (if implemented)  
- Client-side JavaScript for dynamic behaviour
